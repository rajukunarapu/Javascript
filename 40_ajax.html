AjAX = Asynchronous Javascript And Xml

It is a technique, To make asynchronous HTTP Requests from client to server, allowing web pages to be updated dynamically without requiring a full page reload.

That means without any reload the webpage will load.

To make asynchronous http request follows 3 steps:-

step:- 1.create XMLHttpRequest object by using that constructor

step:- 2. by using that variable to create request with open method it takes 5 parameters , open(method,url,asynchronous,username,password)

method= get/post ,get=get the data from the server , post = post the data to the server.

url= "url of the page or if it is in same folder gives the location"

asynchronous = true/false , true = asynchronous programming enabled, false = asynchronous programming not enabled.

username,password related to private api's

step:-3. by using that variable to send methode used to send the xmlhttpRequest to the server.

by using that variable to responseText Methode used to get the response from the server.
