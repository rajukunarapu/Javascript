Events = actions

Types of Events:-

1.mouse:-
1.click
2.double click
3.mouse over
4.mouse out
5.mouse move

2.keyboard :-
1.key press
2.key down = the finger ready to press
3.keyup = the finger in air

3.focus:-
1.focus
2.focus out
3.blur

4.form :-
1.submit
2.reset
3.chage = entering something into input fields.

Who genereate Events:-

1.user :-
1.keypress
2.scroll
3.focus

2.system :-
1.load
2.error
3.abort = mission discontinue.

Types of user generated Events:-

1.browser specific events:- minimize , close , resize , new_window , new_tab.

2.webpage specific events:- scroll-up , scroll-down , click , hover.

How many ways to handle event:-

1.Inline event handler = Add the event in inline of html document element

Syntax:- onEvent_name = "java script code"
Note:- use on before event name

2.using Event properties = get the element by using id and perform events

Syntax:- var_name.onEvent_name = function or java script code

3.using Event Listeners = get the element and use addEventListener('Event_name',fuction name or direct javascript code)
