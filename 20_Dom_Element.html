Dom manipulations on elements:-

1. create an empty element.
2. create an content to element.
3.append the element to html
4.adding element before to another element.
5.remove the element.
6.remove child element.
7.replace the element with another element.

Methods:

1.createElement = create new empty element.
2.innerText = create text for newly added element.
3.appendChild = add the child in last to the parent.
4.remove = remove the given element from its parent.
5.removeChild = remove its child from the parent.
6.replaceWith = replace with the another element.
