Promises:-

Promises are used to handle the call back hell, Promises is an object that produce either resolved, rejection and pending value.

3 states :- 1. reject = rejection operation, 2. pending = operation is not yet completed. 3. fulfilled = operation complete successfull.

Using Promises:- 1.then() = when the promise is fulfilled executes that.

2.catch() = when the promise is reject executes that.

3.finally() = regardless of fulfilled or reject it executes.
